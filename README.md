# SQL Business Analysis Projects

## Overview

This repository contains business-focused SQL data analysis projects completed as part of the AnalystLab Africa Internship Program.

The projects demonstrate practical SQL skills used in real-world business intelligence and analytics, including:

* Data querying
* Aggregation and filtering
* Joins and relationships
* Window functions
* Ranking analysis
* Revenue analysis
* Customer segmentation
* Query optimization
* Business insight generation

---

# Projects Included

## 1. Chinook Database Analysis

Business analysis project using the Chinook Music Store relational database.

### Key Analysis Areas

* Customer purchase analysis
* Top-selling genres
* Artist revenue performance
* Monthly revenue trends
* Customer spending rankings
* Above-average customer segmentation
* Query optimization using indexes

### SQL Skills Demonstrated

* JOIN operations
* GROUP BY and aggregation
* Subqueries
* Window functions
* RANK()
* Index creation
* Revenue analysis

### Business Insights

* Rock was the highest-performing genre
* Iron Maiden generated the highest artist revenue
* Revenue remained relatively stable over time
* High-value customers were identified for retention strategies

---

## 2. Sales Dataset Analysis

SQL analysis project using a transactional sales dataset containing over 2,800 records.

### Key Analysis Areas

* Revenue performance
* Top countries by sales
* Product line analysis
* Monthly sales trends
* Customer revenue rankings
* Order status analysis
* Best-performing products

### SQL Skills Demonstrated

* Aggregation functions
* Window functions
* PARTITION BY
* Ranking functions
* Revenue trend analysis
* Customer segmentation
* Business intelligence reporting

### Business Insights

* Total revenue exceeded $10 million
* The United States was the top-performing market
* Classic Cars generated the highest product revenue
* November consistently recorded peak sales performance
